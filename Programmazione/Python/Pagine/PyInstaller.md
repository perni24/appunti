# PyInstaller

## Sintesi

**PyInstaller** impacchetta applicazioni Python in eseguibili distribuibili, includendo interprete, dipendenze e file necessari.

## Quando usarlo

- Tool desktop o CLI per utenti senza Python installato.
- Distribuzione interna.
- Script operativi da consegnare come eseguibile.

## Come funziona

### Uso base
```powershell
pyinstaller app.py
pyinstaller --onefile app.py
```
