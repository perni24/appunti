# Jupyter

## Sintesi

**Jupyter** permette di lavorare con notebook interattivi che combinano codice, output, testo e visualizzazioni.

## Quando usarlo

- Analisi esplorativa.
- Prototipi.
- Report tecnici.
- Data science.
- Didattica.

## Come funziona

### Buone pratiche
- Eseguire notebook dall'inizio alla fine.
- Spostare logica riusabile in moduli `.py`.
- Versionare dati di input o descrivere come ottenerli.
- Pulire output pesanti se non servono.
