# uv, pipx e poetry

## Sintesi

`uv`, `pipx` e `poetry` sono strumenti dell'ecosistema Python per gestire ambienti, dipendenze, comandi e pacchetti.

## Concetto chiave

- `uv`: tool moderno e veloce per ambienti, dipendenze e lockfile.
- `pipx`: installa applicazioni Python isolate a livello utente.
- `poetry`: gestisce dipendenze, packaging e pubblicazione usando `pyproject.toml`.

## Uso pratico

```powershell
uv venv
uv pip install requests
pipx install black
poetry add fastapi
```

## Quando usarli

- Usa `uv` per workflow veloci e ambienti riproducibili.
- Usa `pipx` per CLI Python globali.
- Usa `poetry` quando vuoi un workflow integrato per librerie e applicazioni.
