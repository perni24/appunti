## Teoria

### Basi
-   Cos'è YAML e Filosofia (YAML Ain't Markup Language)
-   Confronto: YAML vs JSON vs XML
-   Struttura del file e Indentazione
-   Case Sensitivity e Commenti
-   Documenti Multipli (--- e ...)

### Tipi di Dato Scalari
-   Stringhe (Plain, Single, Double Quoted)
-   Numeri (Integer, Float, Hex, Binary)
-   Booleani e Null
-   Date e Timestamp

### Collezioni e Strutture
-   Liste / Sequenze (Block e Flow style)
-   Dizionari / Mappe (Block e Flow style)
-   Collezioni nidificate
-   Set e Ordered Maps (Tipi complessi)

### Gestione Stringhe e Blocchi
-   Literal Block Scalar (|)
-   Folded Block Scalar (>)
-   Block Chomping Indicators (+, -)

### Sintassi Avanzata
-   Ancore (&) e Alias (*)
-   Merge Keys (<<)
-   Tag espliciti (!!str, !!int, etc.)
-   Gestione dei tipi personalizzati

### Qualità e Tooling
-   Schema YAML e Validazione
-   Linter (yamllint)
-   Sicurezza e Parsing (Norway Problem)
-   Integrazione negli Editor (VS Code)
-   Librerie (PyYAML, js-yaml)
