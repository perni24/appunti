**Barrier** è un software open-source che consente di condividere tastiera e mouse tra più computer, anche con diversi sistemi operativi (ad esempio, Windows, macOS, Linux). Funziona tramite rete locale, eliminando la necessità di hardware KVM (Keyboard, Video, Mouse). Supporta il copia-incolla di testo e file tra dispositivi.
