# Log Raw

Questo file registra i documenti inseriti in `Raw/` che sono stati analizzati e assimilati negli appunti.

Usa questo formato per ogni documento completato:

```md
## YYYY-MM-DD - nome-file-originale.ext

- Stato: completato
- File raw: `Raw/completato - nome-file-originale.ext`
- Assimilato in:
  - [[Percorso/Nota]]
- Sintesi: breve descrizione di cosa e stato estratto o integrato.
```
