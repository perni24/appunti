## 1. Fondamenta Matematiche e Statistiche
- **Algebra Lineare:** Matrici, Vettori, Tensori, Decomposizioni (SVD, PCA).
- **Calcolo Multivariata:** Gradienti, Derivate parziali, Backpropagation math.
- **Probabilità e Statistica:** Distribuzioni, Inferenza Bayesiana, Test d'ipotesi.
- **Ottimizzazione:** Gradient Descent, Stochastic Gradient Descent (SGD), Adam.

## 2. Machine Learning (Classico)
- **Supervised Learning:** Regressione Lineare/Logistica, Decision Trees, SVM.
- **Unsupervised Learning:** Clustering (K-Means), Dimensionality Reduction.
- **Ensemble Methods:** Random Forest, Gradient Boosting (XGBoost, LightGBM).
- **Valutazione Modelli:** Cross-Validation, Precision/Recall, F1-Score, Curve ROC.

## 3. Deep Learning (Reti Neurali)
- **Neural Network Architecture:** Perceptron, Multi-Layer Perceptron (MLP).
- **Funzioni di Attivazione:** ReLU, Sigmoid, Tanh, Softmax.
- **Computer Vision (CV):** Convolutional Neural Networks (CNN), Object Detection, Segmentation.
- **Sequence Modeling:** Recurrent Neural Networks (RNN), LSTM, GRU.
- **General Concepts:** Overfitting/Underfitting, Regularization (Dropout, L2).

## 4. Natural Language Processing (NLP)
- **Pre-processing:** Tokenization, Stemming, Lemmatization, Stop-words.
- **Embeddings:** Word2Vec, GloVe, FastText.
- **Transformer Architecture:** Attention Mechanism, Encoder-Decoder, BERT, GPT.
- **Language Tasks:** Sentiment Analysis, Translation, Named Entity Recognition (NER).

## 5. Generative AI (L'era dei LLM)
- **Large Language Models (LLM):** Architetture, Context Window, Tokenization.
- **Prompt Engineering:** Zero-shot, Few-shot, Chain-of-Thought (CoT).
- **RAG (Retrieval-Augmented Generation):** Vector Databases, Semantic Search.
- **Generazione Immagini:** Generative Adversarial Networks (GAN), Diffusion Models (Stable Diffusion).
- **Fine-tuning:** PEFT, LoRA, QLoRA.

## 6. Reinforcement Learning (RL)
- **Fundamentals:** Agent, Environment, State, Action, Reward.
- **Markov Decision Processes (MDP).**
- **Algoritmi:** Q-Learning, Deep Q-Networks (DQN), PPO.
- **Applicazioni:** Robotica, Game AI, Sistemi di controllo.

## 7. MLOps e Infrastruttura
- **Gestione Dati:** ETL, Feature Stores, Data Cleaning.
- **Deployment:** Docker, Kubernetes, Serving API (FastAPI, Flask).
- **Tooling:** PyTorch, TensorFlow, Scikit-learn, Hugging Face.
- **Monitoring:** Drift detection, Logging, Model Versioning.

## 8. Etica e Sicurezza
- **Bias & Fairness:** Identificazione e mitigazione dei pregiudizi.
- **Explainable AI (XAI):** Interpretazione delle "Black Box" (SHAP, LIME).
- **AI Safety:** Allineamento, Robustezza e Sicurezza.
