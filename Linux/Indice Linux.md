# Concetti globali

- Architettura del Sistema (Kernel, Shell, Userspace)
- Filesystem Hierarchy Standard (FHS)
- Gestione Utenti, Gruppi e Permessi (chmod, chown, sudo)
- Shell e Scripting (Bash, variabili d'ambiente, pipe, ridirezioni)
- Elaborazione Testi Avanzata (Regex, Sed, Awk)
- Gestione dei Processi (PID, segnali, job control, monitoraggio)
- Gestione della Memoria e Swap
- Package Management (Repository, dipendenze, compilazione)
- [[pagine/Installazione Linguaggi User-space|Installazione Linguaggi User-space (NVM, Pyenv, Rustup)]]
- Storage e File Systems (Partizionamento, LVM, Mounting, /etc/fstab)
- Crittografia e Sicurezza del Dato (LUKS, GPG)
- Networking (Indirizzamento, routing, SSH, Firewall)
- Processo di Boot (Bootloader, Initramfs, GRUB)
- Inizializzazione e Servizi (Systemd, Runlevel, Timer)
- Gestione Hardware (Udev, D-Bus, Driver, lspci, lsusb)
- Virtualizzazione e Container (Namespaces, Cgroups, Docker, KVM)
- Logging e Analisi del Sistema (Journalctl, file di log)
- Sicurezza e Hardening (User limits, SELinux/AppArmor)
- Gestione dei moduli del Kernel (Modprobe, Insmod, Lsmod)
- Automazione e Task Scheduling (Cron, Anacron)
